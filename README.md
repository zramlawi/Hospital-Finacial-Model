# Hospital Project Finance Model

## Overview

This repository contains a project finance model for a hypothetical
$200 million hospital expansion project.

## Objectives

- Build Sources & Uses
- Create operating forecasts
- Model debt service
- Calculate DSCR
- Perform sensitivity analysis

## Transaction Summary

Project Cost: $200M

Debt: $140M (70%)

Equity: $60M (30%)

Debt Term: 25 Years

Interest Rate: 5.25%
